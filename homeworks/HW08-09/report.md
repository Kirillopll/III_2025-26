# HW08-09 Report

## 1. Выбранный датасет

В данной работе использован датасет **EMNIST (split="balanced")** из библиотеки `torchvision`.

EMNIST является расширением датасета MNIST и содержит изображения рукописных символов.  
Каждое изображение имеет размер **28 × 28 пикселей** и представлено в градациях серого.

Количество классов: **47**

Размеры выборок:

- train: **90240**
- validation: **22560**
- test: **18800**

Форма батча:

- `x.shape = [128, 1, 28, 28]`
- `y.shape = [128]`

Диапазон значений пикселей: **[0, 1]**

Для воспроизводимости экспериментов использован **фиксированный seed = 42**.

# 2. Архитектура модели

В работе использована **полносвязная нейронная сеть (MLP)**.

Базовая архитектура:

- входной слой: **784 признака (28 × 28)**
- скрытые слои: **512 → 256 → 128**
- функция активации: **ReLU**
- выходной слой: **47 классов**

В некоторых экспериментах дополнительно использовались:

- **Dropout**
- **Batch Normalization**
- **Early Stopping**

Оптимизаторы:

- **Adam**
- **SGD**

Добавление **BatchNorm** улучшило стабильность обучения и повысило точность модели.
