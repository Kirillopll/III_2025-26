# HW08-09 – PyTorch MLP: регуляризация и оптимизация обучения

## 1. Кратко: что сделано

В работе исследовано влияние регуляризации и параметров оптимизации на качество обучения многослойной нейронной сети (MLP).

Использован датасет EMNIST (split="balanced"), содержащий изображения рукописных цифр и букв.

В части S08 выполнено сравнение:

E1 — базовая модель без регуляризации  
E2 — модель с Dropout  
E3 — модель с BatchNorm  
E4 — лучшая модель с EarlyStopping  

В части S09 исследовано влияние параметров обучения:

O1 — слишком большой learning rate  
O2 — слишком маленький learning rate  
O3 — SGD с momentum и weight decay  

Выбрана лучшая конфигурация модели на основе validation accuracy.

## 4. Базовая модель и обучение

Использована архитектура MLP:

784 → 512 → 256 → 128 → 47

784 — входной слой (Flatten изображения 28×28)  
512, 256, 128 — скрытые слои  
47 — количество классов  

Функция активации:

ReLU

Функция потерь:

CrossEntropyLoss

Оптимизатор:

Adam

learning rate:

1e-3

Batch size:

128

Используемые методы регуляризации:

Dropout  
BatchNorm  
EarlyStopping  

EarlyStopping:

patience = 4  
метрика — validation accuracy

---

## 5. Часть A (S08): регуляризация (E1-E4)

Проведены эксперименты:

E1 — базовая модель без регуляризации  

E2 — добавлен Dropout (p = 0.3)  

E3 — добавлен BatchNorm между Linear и функцией активации  

E4 — лучшая модель (E2 или E3) + EarlyStopping  

Наблюдения:

Dropout уменьшает переобучение модели.

BatchNorm ускоряет обучение и делает процесс обучения более стабильным.

EarlyStopping позволяет остановить обучение на оптимальной эпохе.

Лучшая модель определяется по максимальному значению validation accuracy.

---

## 6. Часть B (S09): LR, оптимизаторы, weight decay (O1-O3)

Проведены эксперименты:

O1 — слишком большой learning rate  
Adam lr = 1e-1  

O2 — слишком маленький learning rate  
Adam lr = 1e-5  

O3 — SGD с momentum и weight decay  
momentum = 0.9  
weight decay = 1e-4  

Наблюдения:

слишком большой learning rate приводит к нестабильности функции потерь

слишком маленький learning rate приводит к очень медленному обучению

SGD с momentum показывает стабильные результаты обучения

weight decay уменьшает переобучение

## 8. Анализ

Без регуляризации модель склонна к переобучению:

train accuracy увеличивается быстрее, чем validation accuracy.

Dropout уменьшает переобучение за счёт случайного отключения нейронов.

BatchNorm ускоряет обучение и делает градиенты более стабильными.

EarlyStopping позволяет остановить обучение при достижении оптимального качества.

Слишком большой learning rate приводит к резким скачкам функции потерь.

Слишком маленький learning rate приводит к очень медленной сходимости.

SGD с momentum показывает стабильное уменьшение функции потерь.

weight decay уменьшает сложность модели и улучшает обобщающую способность.

Оптимальный learning rate около 1e-3.
