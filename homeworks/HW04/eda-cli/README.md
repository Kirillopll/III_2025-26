# HW04 – eda_cli: HTTP-сервис качества датасетов (FastAPI)

Данный проект представляет собой учебный HTTP-сервис для оценки качества
CSV-датасетов.  
Сервис построен поверх EDA-ядра, реализованного в HW03, и демонстрирует,
как CLI-приложение и библиотека могут быть обёрнуты в REST API.

Проект выполнен в рамках курса **«Инженерия ИИ»**, семинары **S03–S04**.

---

## 1. Назначение проекта

Цели проекта:

- реализовать базовый EDA-анализ CSV-файлов;
- вычислять эвристические показатели качества данных;
- предоставить доступ к логике анализа через:
  - CLI (HW03);
  - HTTP-API на FastAPI (HW04);
- закрепить навыки:
  - проектирования REST-эндпоинтов;
  - работы с FastAPI и uvicorn;
  - повторного использования логики ядра (EDA) без дублирования кода.

---

## 2. Используемые технологии

- **Python 3.11**
- **FastAPI** — HTTP-сервис
- **uvicorn** — ASGI-сервер
- **pandas** — обработка данных
- **matplotlib** — визуализация
- **typer** — CLI
- **pytest** — тестирование
- **uv** — управление окружением и зависимостями
